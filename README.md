# Battleship Project

The classic 2 player gridbased game where your objective is two destroy all your opponents ships by strategically guessing the location of their ships.

## Instructions

- Players must place ships on the grid vertically and horizontally

* Each player takes turns guessing battleship location on grid ex. "A 10"

- Each ship has a damage rating 2-5 which alots for the amount of grid space taken (per square)

- Battleship size:
  Carrier: 5
  Battleship: 4
  Cruiser: 3
  Submarine: 3
  Destroyer: 2

- First player to destroy all battleships wins
